## 📊 Key Features

### **Sales Analytics**
- 📈 Revenue trends by product/category  
- 🗓️ Time-series analysis (daily/weekly/monthly)  
- 📊 Geographic sales distribution  

### **Customer Insights**
- 🧑‍🤝‍🧑 RFM (Recency-Frequency-Monetary) segmentation  
- 🛒 Purchase behavior patterns  
- 🔄 Churn prediction model  

### **Inventory Analysis**
- 📦 Stock movement analysis  
- ⚠️ Low-stock alerts  
- 📉 Dead stock identification
